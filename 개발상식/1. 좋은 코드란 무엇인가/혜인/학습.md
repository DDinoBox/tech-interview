# 좋은 코드란 무엇인가?

## 가독성이 좋은 코드
  - 직관적인 변수, 함수, 클래스 이름 짓기
  - 코드가 길어지거나 직관적이지 못한 경우 별도의 메소드로 분리하기

## 기본을 지키는 코드
  - 직관적인 패키지 구조
  - 들여쓰기 뎁스가 최소인 함수
  - 코딩 컨벤션을 지키기
